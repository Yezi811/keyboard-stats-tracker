## ✨ 主要功能

### 1. 实时键盘统计
- 实时记录所有按键操作
- 自动统计按键次数和频率
- 支持所有标准键盘按键（包括数字键盘）

### 2. 数据可视化
- 柱状图展示按键分布
- 详细的按键统计表格
- 百分比显示按键占比

### 3. 时间周期查看
- 按日查看统计数据
- 按月查看统计数据
- 按年查看统计数据

### 4. 搜索功能
- 精确搜索特定按键
- 支持按键名称搜索（如：A, Enter, Space）
- 确认/重置按钮控制

### 5. 数据导出
- 支持导出为 JSON 格式
- 支持导出为 CSV 格式
- 可选择导出时间范围

### 6. 数据管理
- 一键清空所有数据
- 自动备份功能
- 数据恢复功能
- 查看历史备份列表

### 7. 系统集成
- 系统托盘图标
- 最小化到托盘
- 开机自启动选项
- 单实例运行（防止重复启动）

## 🚀 使用方法

### 安装版
1. 双击 `Keyboard Stats Tracker Setup 1.0.0.exe`
2. 按照安装向导提示完成安装
3. 从桌面或开始菜单启动应用

### 便携版
1. 将 `Keyboard Stats Tracker 1.0.0.exe` 复制到任意位置
2. 双击运行即可

## 📊 数据存储位置

- **数据库文件**: `%APPDATA%\keyboard-stats-tracker\keystrokes.db`
- **备份文件**: `%APPDATA%\keyboard-stats-tracker\backups\`
- **日志文件**: `%APPDATA%\keyboard-stats-tracker\error.log`

## 🔍 搜索功能使用

1. 在搜索框中输入按键名称（如：A, Enter, Space, Backspace）
2. 点击"确认"按钮应用搜索
3. 点击"重置"按钮清除搜索并显示所有数据

**支持的按键名称示例**:
- 字母: A, B, C, ... Z
- 数字: 0, 1, 2, ... 9
- 功能键: Enter, Space, Backspace, Tab, Escape
- 数字键盘: Numpad0, Numpad1, ... Numpad9
- 特殊键: Shift, Ctrl, Alt, CapsLock
- 方向键: ArrowUp, ArrowDown, ArrowLeft, ArrowRight

## 💾 数据导出

1. 选择时间周期（日/月/年）
2. 选择导出格式（JSON/CSV）
3. 点击"导出数据"按钮
4. 选择保存位置

## 🗑️ 数据清空与恢复

### 清空数据
1. 点击"清空数据"按钮
2. 确认操作
3. 系统会自动创建备份

### 恢复数据
1. 点击"恢复数据"按钮
2. 从备份列表中选择要恢复的备份
3. 确认恢复操作

## ⚙️ 系统托盘功能

右键点击系统托盘图标可以：
- 显示/隐藏主窗口
- 设置开机自启动
- 退出应用
