# 键盘统计追踪器 - 用户使用指南

## 📥 安装

### 便携版（推荐）
1. 下载 ZIP 文件
2. 解压到任意文件夹（建议：`C:\Program Files\Keyboard Stats Tracker`）
3. 双击 `Keyboard Stats Tracker.exe` 运行
4. 首次运行可能需要允许防火墙访问

### 安装程序版
1. 下载安装程序（.exe 文件）
2. 双击运行安装程序
3. 选择安装位置
4. 完成安装
5. 从开始菜单或桌面快捷方式启动

## ✨ 功能介绍

### 1. 实时键盘统计
- 自动记录每次按键
- 实时显示总敲击次数
- 按键详细统计表格

### 2. 数据可视化
- **日统计**：查看某一天的按键分布（柱状图）
- **月统计**：查看整月的按键趋势（折线图）
- **年统计**：查看全年的按键趋势（折线图）

### 3. 数据导出
- **JSON 格式**：适合程序处理
- **CSV 格式**：适合 Excel 打开
- 导出内容：按键名称、次数、占比

### 4. 系统托盘
- 最小化到系统托盘
- 后台持续统计
- 双击托盘图标恢复窗口

### 5. 开机自启动
- 可选择开机自动启动
- 后台静默运行
- 不影响系统性能

### 6. 数据管理
- **备份**：重置前自动备份
- **恢复**：从备份恢复数据
- **重置**：清空所有统计数据

## 🎮 使用方法

### 基本操作

#### 查看统计数据
1. 选择统计周期：日/月/年
2. 选择日期
3. 查看图表和详细数据

#### 导出数据
1. 选择要导出的日期和周期
2. 点击"导出 JSON"或"导出 CSV"
3. 选择保存位置
4. 完成导出

#### 刷新图表
- 总数和表格会自动实时刷新
- 图表需要手动刷新（点击"刷新图表"按钮）
- 避免图表频繁跳动影响观感

### 高级操作

#### 最小化到托盘
- 点击窗口的最小化按钮
- 或点击关闭按钮
- 应用会隐藏到系统托盘
- 双击托盘图标恢复窗口

#### 设置开机自启动
1. 右键点击托盘图标
2. 勾选"开机自启动"
3. 下次开机会自动启动

#### 完全退出应用
1. 右键点击托盘图标
2. 选择"退出"
3. 应用完全关闭

#### 重置数据
1. 点击"重置数据"按钮
2. 确认操作
3. 系统会自动创建备份
4. 所有数据被清空

#### 恢复数据
1. 点击"恢复数据"按钮
2. 选择要恢复的备份
3. 确认恢复
4. 数据恢复完成

## ❓ 常见问题

### Q: 杀毒软件报警怎么办？
**A:** 这是误报。应用未进行代码签名，部分杀毒软件会警告。
- 解决方法：添加到杀毒软件白名单
- 或选择"仍要运行"

### Q: 如何开机自启动？
**A:** 右键托盘图标 → 勾选"开机自启动"

### Q: 数据会丢失吗？
**A:** 不会。数据实时保存到数据库，即使突然关机也不会丢失。

### Q: 占用多少资源？
**A:** 
- 内存：约 100-150 MB
- CPU：几乎为 0（待机时）
- 磁盘：数据库文件约 1-10 MB

### Q: 如何卸载？
**A:** 
- **便携版**：直接删除文件夹即可
- **安装版**：
  1. 打开"设置" → "应用"
  2. 找到"Keyboard Stats Tracker"
  3. 点击"卸载"

### Q: 会记录密码吗？
**A:** 不会。应用只记录按键次数，不记录具体内容。

### Q: 可以在多台电脑使用吗？
**A:** 可以。每台电脑的数据独立存储。

### Q: 如何备份数据到其他电脑？
**A:** 
1. 导出数据为 JSON/CSV
2. 或复制整个数据文件夹
3. 在新电脑上导入或替换

### Q: 打开多个窗口会重复统计吗？
**A:** 不会。应用有单实例锁定，只能运行一个实例。

### Q: 图表为什么不自动更新？
**A:** 为了避免图表频繁跳动影响观感，图表需要手动刷新。

### Q: 如何查看历史数据？
**A:** 选择对应的日期和周期即可查看。
